# This is the README.md file for the **github-final-project**

A calculator that calculates simple interest given principal, annual rate of interest and time period in years.

Input:  
- p, principal amount  
- t, time period in years  
- r, annual rate of interest  

Output  
simple interest = p * t * r
``` :contentReference[oaicite:0]{index=0}

In short, this project is a **simple interest calculator** that takes the principal amount, time (in years), and the annual interest rate, and outputs the calculated simple interest. :contentReference[oaicite:1]{index=1}
